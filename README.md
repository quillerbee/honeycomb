## How to run locally?

1. `npm run start`
1. `npm run serve`
1. Ratel: http://127.0.0.1:8000
1. GraphQL Introspection: http://127.0.0.1:8080/graphql
1. GraphQL Voyager: Paste the introspection SDL from GraphQL Playground
